# 🎨 Fastfetch Theme Selector

A simple interactive script to select and apply **Fastfetch themes** using `fzf` and `gum`.

---

## ✨ Features

- 🔥 Interactive theme selection
- ⚡ Fast and lightweight
- 🎯 Easy to use with shortcuts
- 🧩 Fully customizable

---

## 📦 Requirements

Make sure you have the following installed:

- `fastfetch`
- `fzf`
- `gum`

### Arch Linux:
```bash
sudo pacman -S fastfetch fzf gum
